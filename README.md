# Transcribe Call

A Claude Code skill for recording and transcribing meetings/calls using local transcription (Parakeet MLX).

## Documentation

See [SKILL.md](./SKILL.md) for complete documentation and usage instructions.

## Installation

```bash
npx skills add HartreeWorks/skill--transcribe-call
```

Or install all HartreeWorks skills: `npx skills add HartreeWorks/skills`
